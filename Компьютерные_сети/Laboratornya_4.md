# Отчет по лабораторной работе
## Подключение к Raspberry Pi по протоколу SSH

### Часть 1. Подключение и работа с файловой системой

1. Выполнено подключение к Raspberry Pi по SSH:
   ```bash
   ssh user@192.168.129.150
   ```
   Пароль: `123`

2. В корневом каталоге `/` найден каталог группы, внутри создан каталог с фамилией студента.

3. Создана структура каталогов и файлов согласно заданию.

4. Выполнена команда для отображения структуры:
   ```bash
   ls -R
   ```

5. Завершение сеанса:
   ```bash
   exit
   ```

### Часть 2. Настройка SSH-ключей для входа без пароля

1. Проверка существующих ключей:
   ```bash
   ls ~/.ssh
   ```

2. Генерация новой пары SSH-ключей:
   ```bash
   ssh-keygen
   ```
   - Созданы файлы: `id_rsa` (приватный ключ) и `id_rsa.pub` (публичный ключ)

3. Копирование публичного ключа на Raspberry Pi:
   ```bash
   ssh-copy-id user@192.168.129.150
   ```

4. Проверка подключения без пароля выполнена успешно.

### Результат

- Установлено SSH-соединение с Raspberry Pi
- Создана необходимая структура каталогов
- Настроен вход по SSH-ключам (без ввода пароля)
