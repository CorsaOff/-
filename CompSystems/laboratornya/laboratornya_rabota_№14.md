## Групповые политики Windows 10

## Теоретическая часть

### 1. Запуск редактора локальной групповой политики

Чтобы запустить редактор локальной групповой политики, выполните следующие действия:

1. Щелкните на кнопке **Пуск**.
2. Введите `gpedit.msc`.
3. Нажмите клавишу **Enter**.

Открывающееся при этом окно **Редактор локальной групповой политики** показано на рис. 9.1. Слово *локальных* отражает тот факт, что выполняется редактирование групповых политик на вашем собственном, а не на удаленном компьютере.

> **Рис. 9.1.** Редактор локальной групповой политики предназначен для изменения групповых политик на локальном ПК.

---

### 2. Работа с групповыми политиками

Окно **Редактор локальной групповой политики** разделено на две части.

*   **Левая панель** — содержит древовидную иерархическую структуру категорий политик, разделенных на две основные категории:
    *   **Конфигурация компьютера** — применяются ко всем пользователям, выполняются перед входом в систему.
    *   **Конфигурация пользователя** — применяются только к текущему пользователю, не применяются до тех пор, пока пользователь не войдет в систему.
*   **Правая панель** — содержит политики той категории, которая выбрана в левой панели.

**Основная идея:** Открытие ветвей дерева для отыскания нужной категории. При щелчке на категории ее политики отображаются в правой панели.

> **Рис. 9.2.** При выборе категории в левой панели политики этой категории отображаются в правой панели.

**Столбцы правой панели:**
*   **Параметр** — отображает имя политики.
*   **Состояние** — текущее состояние данной политики.

> **Примечание:** Если описание не отображается, щелкните на вкладке **Расширенный** (рис. 9.3).

---

### 3. Настройка политики

Чтобы настроить политику, **дважды щелкните** на ней. Тип отображаемого окна зависит от политики.

*   **Для простых политик** (рис. 9.4) доступно одно из трех состояний:
    *   **Не задано** — политика не действует.
    *   **Включить** — политика действует, настройки активизированы.
    *   **Отключить** — политика действует, параметры отключены.

*   **Другие политики** требуют дополнительной информации. Например, на рис. 9.5 показано окно для политики *Элементы, отображаемые в панели мест*. Когда опция **Включить** активизирована, текстовые поля становятся доступными для ввода путей к папкам.

> **Рис. 9.5.** Более сложные политики требуют дополнительной информации (например, список папок для отображения в панели мест).

---

### 4. Фильтрация политик

> Существует около 3000 политик. Попытка найти нужную политику путем блуждания по ветвям подобна поиску иголки в стогу сена.

**Улучшенная функция фильтрации** позволяет усечь ландшафт доступных политик до приемлемого размера.

**Пример поиска политики "Элементы, отображаемые в панели мест":**

1.  Выберите ветвь:
    `Конфигурация пользователя` → `Административные шаблоны` → `Все параметры`
2.  Выберите в меню **Действия** → **Параметры фильтра**, чтобы открыть диалоговое окно.
3.  Удостоверьтесь, что флажок **Включить фильтры по ключевым словам** отмечен.
4.  В текстовом поле **Фильтры по словам** введите слово или словосочетание (например, *"мест"*).
5.  В раскрывающемся списке выберите уровень соответствия:
    *   **Любой** — содержит хотя бы один термин.
    *   **Все** — содержит все термины.
    *   **Точный** — полное совпадение.
6.  Используйте флажки **В**, чтобы указать место поиска:
    *   **Заголовок параметра политики**
    *   **Текст справки**
    *   **Комментарий**
7.  Щелкните **OK**.

После определения фильтра выберите **Действие** → **Фильтр** (или кнопку **Фильтр** на панели инструментов). Редактор отобразит только соответствующие политики.

> **Рис. 9.6** — Параметры фильтра.
> **Рис. 9.7** — Результат включения фильтра.

## Задание

### Примеры групповых политик

#### Настройка окна безопасности Windows

При нажатии **Ctrl+Alt+Delete** открывается окно безопасности Windows с кнопками:

*   **Блокировать компьютер** — скрывает рабочий стол (или используйте `Win + L`).
*   **Сменить пользователя** — открывает сеанс другого пользователя.
*   **Выйти из системы** — открывает экран приветствия.
*   **Сменить пароль** — позволяет сменить пароль.
*   **Запустить диспетчер задач** — открывает Диспетчер задач.

**Как удалить кнопки из окна безопасности Windows:**

1.  Откройте **Редактор локальной групповой политики**.
2.  Перейдите к ветви:
    `Конфигурация пользователя` → `Административные шаблоны` → `Система` → `Варианты действий после нажатия CTRL+ALT+DEL`
3.  Дважды щелкните на нужной политике:
    *   *Запретить изменение пароля* (отключает кнопку "Сменить пароль")
    *   *Запретить блокировку компьютера* (отключает "Блокировать компьютер")
    *   *Удалить диспетчер задач* (отключает "Запустить диспетчер задач")
    *   *Запретить завершение сеанса* (отключает "Выйти из системы")
4.  В диалоговом окне щелкните **Включить** → **OK**.

> **Рис. 9.9.** Окно безопасности Windows с удаленными кнопками.

---

#### Настройка панели мест

Левая часть диалоговых окон **Сохранить как** и **Открыть** (в старом стиле) содержит ссылки на часто используемые места: *Недавние места, Рабочий стол, Библиотеки, Компьютер, Сеть* (рис. 9.10).

**Как настроить панель мест через GPO:**

1.  Откройте **Редактор локальной групповой политики**.
2.  Перейдите к ветви:
    `Конфигурация пользователя` → `Административные шаблоны` → `Компоненты Windows` → `Проводник Windows` → `Общее диалоговое окно открытия файлов`
3.  Дважды щелкните на политике **Элементы, отображаемые в панели мест**.
4.  Щелкните **Включить**.
5.  В полях **Элемент 1 – Элемент 5** введите пути к папкам (локальным или сетевым).
6.  Нажмите **OK**.

> **Рис. 9.11.** Диалоговое окно с пользовательской панелью мест.

**Скрыть панель мест:**
В той же ветви дважды щелкните на политике **Скрыть панель адресов из общих диалогов открытия файлов** → **OK**.

---

#### Увеличение размера списка недавних документов

По умолчанию в меню **Пуск** → **Недавние документы** отображается 15 документов.

**Как изменить размер списка:**

1.  Откройте **Редактор локальной групповой политики**.
2.  Перейдите к ветви:
    `Конфигурация пользователя` → `Административные шаблоны` → `Компоненты Windows` → `Проводник Windows`
3.  Дважды щелкните на политике **Максимальная длина списка "Недавние документы"**.
4.  Щелкните **Включить**.
5.  Счетчиком укажите количество документов (от **1** до **9999**).
6.  Нажмите **OK**.

> **Примечание:** Если указано больше документов, чем помещается на экране, Windows 7 добавляет кнопки прокрутки.

#### Включение диалога слежения за завершением работы

Позволяет документировать причины завершения работы (запланированное/незапланированное, комментарии).

**Как включить через GPO:**

1.  Откройте **Редактор локальной групповой политики**.
2.  Перейдите к ветви:
    `Конфигурация компьютера` → `Административные шаблоны` → `Система`
3.  Дважды щелкните на политике **Отображать диалог слежения за завершением работы**.
4.  Щелкните **Включить**.
5.  В списке выберите опцию **Всегда**.
6.  Нажмите **OK**.

> **Рис. 9.12.** Диалоговое окно "Завершение работы Windows".

**Элементы управления диалога:**
*   **Запланированное** — флажок, указывающий на плановость завершения.
*   **Причина** — список причин (зависит от флажка "Запланированное").
*   **Комментарий** — текстовое поле для описания (обязательно для опции "Другое").

### Дополнительно: Реестр (Альтернативные методы)

**Удаление кнопок окна безопасности (HKCU):**
*   `HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System`
    *   `DisableChangePassword`
    *   `DisableLockWorkstation`
    *   `DisableTaskMgr`
*   `HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer`
    *   `NoLogoff = 1` (удалить кнопку "Выйти из системы")

**Настройка панели мест (HKCU):**
1.  Перейдите к `HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\`
2.  Создайте раздел `comdlg32` → внутри него создайте `Placesbar`
3.  Создайте строковые параметры `Place0` – `Place4` с путями к папкам.

**Диалог слежения за завершением работы (HKLM):**
*   `HKLM\Software\Policies\Microsoft\Windows NT\Reliability`
    *   `ShutdownReasonOn = 1`
    *   `ShutdownReasonUI = 1`
