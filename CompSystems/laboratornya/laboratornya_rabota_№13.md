## Тема: Установка и настройка центра обновления Windows

**Цель:** Изучить принципы настройки и обновления ОС Windows.

---

## 1. Теоретические сведения

Обновления Windows делятся на несколько категорий:

* **Важные обновления:** Обеспечивают значительное улучшение защиты, безопасности и надежности. Устанавливаются автоматически.
* **Рекомендуемые обновления:** Устраняют некритические проблемы и улучшают работу программ. Могут устанавливаться автоматически.
* **Необязательные обновления:** Драйверы или программное обеспечение от Microsoft. Устанавливаются вручную.
* **Основные типы по классификации:**
    * *Обновление безопасности:* Исправление уязвимостей.
    * *Критические обновления:* Исправление критических ошибок, не связанных с безопасностью.
    * *Пакеты обновлений (Service Packs):* Сборные наборы всех предыдущих исправлений и изменений в дизайне.

### Этап 2. Отключение обновлений через Групповые политики

1. Нажмите `Win + R`, введите `gpedit.msc` и нажмите Enter.
2. Перейдите по пути:
   `Конфигурация компьютера` — `Административные шаблоны` — `Компоненты Windows` — `Центр обновления Windows`.
3. Найдите параметр **«Настройка автоматического обновления»**, откройте его и установите значение **«Отключено»**.
4. Проверьте изменения в Центре обновления Windows и сделайте скриншот/запись в отчете.

### Этап 3. Ручная установка обновлений
1. Откройте проводник и в строке пути введите: `\\kbastrikin` (для копирования обновлений из сетевой папки).
2. Установите обновления вручную.
3. Используйте сайт [Microsoft Update Catalog](http://www.catalog.update.microsoft.com) для поиска информации по номеру KB.
4. Заполните таблицу:

| Номер обновления | Классификация | Версия | Размер |
| :--- | :--- | :--- | :--- |
| KB5084066 | Security and Quality Rollup | 1809 | от 96,7 МБ до 173,1 МБ |
| KB5082123 | Security Update | 1809 | 812,8 МБ |

### Этап 4. Настройка службы «Центр обновления»
1. Перейдите: **Панель управления -> Администрирование -> Управление компьютером -> Службы**.
2. Найдите службу **«Центр обновления Windows»**, откройте её свойства.
3. На вкладке **Общие** установите тип запуска: **«Вручную»**.
4. На вкладке **Восстановление** настройте параметры перезапуска системы при сбое согласно указаниям преподавателя.
5. Сохраните настройки.

# Ответы на вопросы по Windows Update

### Этап 5. Контрольные вопросы

### 1. Что такое обновление?
**Обновление** — это программный пакет, который вносит изменения в установленную операционную систему или приложения. Его главные задачи:
*   Устранение критических уязвимостей (безопасность).
*   Исправление ошибок и багов (стабильность).
*   Оптимизация производительности.
*   Добавление новых функций.

### 2. Типы обновлений Windows
Windows разделяет обновления на несколько ключевых категорий:
1.  **Обновления качества (Quality Updates):** Включают в себя патчи безопасности и исправления критических ошибок. Выходят регулярно («Вторник патчей»).
2.  **Обновления компонентов (Feature Updates):** Крупные апдейты, которые выходят раз в год. Фактически это новая версия системы с новыми возможностями.
3.  **Обновления драйверов:** Улучшают совместимость системы с видеокартами, звуковыми платами и другой периферией.
4.  **Обновления определений:** Обновление баз данных для «Защитника Windows» (антивируса) для распознавания новых угроз.

### 3. Для чего нужен Центр обновления Windows?
**Центр обновления Windows** — это встроенный сервисный механизм, который автоматизирует процесс обслуживания системы. Он нужен для:
*   Поиска и загрузки актуального ПО в фоновом режиме.
*   Контроля за целостностью и безопасностью ОС.
*   Установки исправлений без необходимости ручного поиска файлов на сайтах Microsoft.
*   Обеспечения корректной работы «железа» через актуальные драйверы.
