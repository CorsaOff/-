# Lab35


# Управление файлами и директориями в GNU/Linux с помощью Midnight Commander

## Цель работы
Освоить файловый менеджер Midnight Commander для работы с файловой системой в ОС GNU/Linux.

## Рассматриваемые вопросы
1. Первое знакомство с интерфейсом Midnight Commander.
2. Настройка внешнего вида панелей и отображения содержимого.
3. Способы сортировки файлов и папок.
4. Просмотр и редактирование текстовых файлов, создание каталогов.
5. Копирование файлов и папок.
6. Перемещение файлов и папок.
7. Удаление объектов.
8. Создание символических ссылок.
9. Просмотр информации о занятом и свободном месте.
10. Подключение к удалённым FTP-серверам.
11. Подключение по SSH через SFTP.
12. Использование командной строки внутри программы.

## Ход выполнения

> **Примечание:** Практикум проводится на компьютерах с ОС Canonical GNU/Linux Ubuntu 12.04 LTS или Debian 7 и графической средой GNOME.

### Начало работы
Включите компьютер и войдите в систему, используя свои учётные данные.

## 1. Запуск Midnight Commander

Откройте терминал комбинацией `<Ctrl+Alt+T>` или через меню `Applications → Accessories → Terminal`.  
Введите команду `mc` и нажмите `Enter`. Разверните окно на весь экран.

Изучите панели, меню и клавиши быстрого доступа.

Для подготовки отчёта используйте LibreOffice Writer (`Applications → Office → LibreOffice Writer`).

Сделайте скриншот окна Midnight Commander и вставьте в отчёт.

## 2. Настройка режимов отображения

### Древо каталогов
В меню `Right` выберите `Tree` (`<F9>`, затем `<R>`, потом `<T>`). Изучите структуру папки `/etc`. Сохраните скриншот.

### Свойства объекта
В меню `Right` выберите `Info` (`<F9>`, `<R>`, `<Ctrl+x>`, `<i>`). Выделите любой файл или каталог на левой панели — справа появится информация о нём. Сделайте скриншот.

### Быстрый просмотр
В меню `Right` выберите `Quick View` (`<F9>`, `<R>`, `<Ctrl+x>`, `<q>`). Откройте на левой панели файл, например `/etc/fstab` — справа увидите его содержимое. Зафиксируйте скриншотом.

### Возврат к обычному списку
В меню `Right` выберите `File listing` (`<F9>`, `<R>`, `<g>`). Сделайте снимок.

## 3. Сортировка объектов

### По размеру
`Right → Sort order` (`<F9>`, `<R>`, `<S>`). Выберите `Size` (`<S>`) и нажмите `OK`. Перейдите в `/etc` — самый большой файл окажется внизу. Скриншот.

### По времени изменения
Там же выберите `Modify Time` (`<m>`). Внизу списка будет файл, изменённый позже всех. Скриншот.

### По имени
Выберите `Name` (`<N>`). Внизу окажется файл с именем, близким к концу алфавита. Скриншот.

## 4. Просмотр, редактирование, создание папок

- **Просмотр файла:** выделите файл (например, `/etc/fstab`) и нажмите `<F3>` (`File → View`). Скриншот.
- **Создание файла:** перейдите в домашний каталог, нажмите `<Shift+F4>`, напишите текст, сохраните как `newfile` (`<F2>`), выйдите (`<F10>`). Откройте снова по `<F4>`. Скриншот.
- **Создание папки:** нажмите `<F7>` (`File → Mkdir`), введите имя `new_dir`. Поставьте курсор на папку — скриншот.

## 5. Копирование

> **Совет:** Левая панель — исходная папка, правая — целевая. Переключайтесь мышью или `<Tab>`.

Скопируйте каталог `/etc` в `/tmp`:
- На левой панели откройте `/`, на правой — `/tmp`.
- Выберите папку `etc` и нажмите `<F5>`.
- В диалоге можно отметить `Preserve attributes` (аналог `cp -p`).
- Нажмите `OK`.

После копирования перейдите на правую панель в скопированный каталог и сделайте скриншот.

## 6. Перемещение и переименование

Откройте на обеих панелях `/tmp`. Выделите папку `etc` и нажмите `<F6>` (`File → Rename/Move`).  
Папка переместится на правую панель. При желании переименуйте её в `etc_new`.

## 7. Удаление

Выделите ненужный объект (например, `/tmp/etc_new`) и нажмите `<F8>` (или `File → Delete`).  
Подтвердите удаление:
- сначала `Yes`,
- затем для рекурсивного удаления всей папки выберите `All`.

Сделайте скриншот после удаления.

## 8. Поиск

Откройте меню `Command → Find file` (`<F9>`, `<C>`, `<F>`).

### Поиск файлов по маске
Найдите все изображения `.png` в домашней папке:
- В поле введите `*.png`
- Снимите галочку `Search for content`
- Запустите поиск

После завершения сделайте скриншот со списком найденных файлов.

### Поиск по содержимому
Найдите текст `clock` внутри файлов каталога `/etc`:
- В поле `File name:` поставьте `*`
- Отметьте `Search for content`
- В поле `Content:` введите `clock`
- Запустите поиск

Сделайте скриншот результатов.

## 9. Создание символических ссылок

1. Перейдите в `/tmp` на обеих панелях.
2. Выделите любой файл.
3. Нажмите `<F9>`, затем `<F>`, потом `<S>` (или `File → Symlink`).
4. В строке `Symbolic link filename:` добавьте в конец имени `_link` и подтвердите.

Перед именем ссылки появится значок `@` — сделайте скриншот.  
Наведите курсор на ссылку — внизу панели отобразится путь к исходному файлу.

## 10. Просмотр информации о дисковом пространстве

### Быстрая информация
В правом нижнем углу каждой панели отображается:
- доступное место
- общий объём (в байтах и процентах)

### Размер папок
1. Выберите `Command → Show directory sizes` (или нажмите `<Ctrl+Space>`)
2. Зайдите в `/etc` и отсортируйте содержимое по размеру
3. Выполните команду — увидите реальный занимаемый объём каждой папки

Сделайте скриншот.

## 11. Подключение к FTP-серверу

Midnight Commander имеет встроенный FTP-клиент:

1. Откройте `Left → FTP link` (`<F9>`, `<L>`, `<P>`)
2. В поле введите `ftp://mirror.yandex.ru`
3. Нажмите `OK`

На активной панели появится содержимое удалённого сервера. Сделайте скриншот.

## 12. Подключение по SSH (SFTP)

1. Откройте `Left → Shell link` (`<F9>`, `<L>`, `<S>`)
2. Введите адрес сервера (например, `server`)
3. Нажмите `OK`

На панели отобразится корневой каталог удалённого сервера. Сделайте скриншот.

## 13. Использование встроенной командной строки

В нижней части окна Midnight Commander находится командная строка.

1. Введите команду `date` и нажмите `Enter`
2. Нажмите `<Ctrl+O>` — Midnight Commander временно свернётся, и вы увидите вывод команды
3. Сделайте скриншот
4. Вернитесь обратно тем же `<Ctrl+O>`

## Заключение

В ходе практической работы вы освоили основные возможности Midnight Commander:
- навигацию по файловой системе
- операции с файлами и каталогами (копирование, перемещение, удаление, создание)
- поиск файлов
- работу с символическими ссылками
- подключение к удалённым серверам по FTP и SSH
- использование встроенной командной строки

Полученные навыки позволят вам эффективно управлять файлами в среде GNU/Linux как локально, так и удалённо.


# Ответы на вопросы для самопроверки

## 1. Как запустить Midnight Commander?
Открыть терминал (`Ctrl+Alt+T`), ввести команду `mc` и нажать `Enter`.

## 2. Как посмотреть структуру каталога в виде дерева?
`Right → Tree` или нажать `<F9>`, затем `<R>`, потом `<T>`.

## 3. Как вывести подробную информацию о выделенном объекте?
`Right → Info` или `<F9>`, `<R>`, `<Ctrl+x>`, `<i>`.

## 4. Как включить режим быстрого просмотра содержимого файла?
`Right → Quick View` или `<F9>`, `<R>`, `<Ctrl+x>`, `<q>`.

## 5. Каким образом переключить панель в обычный режим показа списка файлов?
`Right → File listing` или `<F9>`, `<R>`, `<g>`.

## 6. Как отсортировать файлы по их размеру?
`Right → Sort order` → выбрать `Size` (клавиша `<S>`) → `OK`.

## 7. Как выполнить сортировку по дате последнего изменения?
`Right → Sort order` → выбрать `Modify Time` (клавиша `<m>`) → `OK`.

## 8. Как отсортировать по алфавиту (имени)?
`Right → Sort order` → выбрать `Name` (клавиша `<N>`) → `OK`.

## 9. Что нажать, чтобы просмотреть файл во встроенном просмотрщике?
Выделить файл и нажать `<F3>` (или `File → View`).

## 10. Какая клавиша или пункт меню отвечает за создание нового файла?
`<Shift+F4>` — открывает новый файл для редактирования, после сохранения файл создаётся.

## 11. Как открыть файл для редактирования во встроенном редакторе?
Выделить файл и нажать `<F4>` (или `File → Edit`).

## 12. Как создать новую папку?
Нажать `<F7>` (или `File → Mkdir`), ввести имя папки, нажать `OK`.

## 13. Какими клавишами копировать объекты с левой панели на правую?
Выделить объект на левой панели, нажать `<F5>`.

## 14. Как переместить или переименовать объект, используя две панели?
Выделить объект, нажать `<F6>` (или `File → Rename/Move`).

## 15. Какое действие нужно для удаления файлов и папок?
Выделить объект, нажать `<F8>` (или `File → Delete`), подтвердить удаление.

## 16. Как найти файл по имени (маске)?
`Command → Find file` (`<F9>`, `<C>`, `<F>`), в поле ввести маску (например, `*.png`), снять галочку `Search for content`, запустить поиск.

## 17. Как искать определённый текст внутри файлов?
`Command → Find file`, в поле `File name:` поставить `*`, отметить `Search for content`, в поле `Content:` ввести искомый текст, запустить поиск.

## 18. Каким образом создать символическую ссылку?
Выделить файл, нажать `<F9>`, затем `<F>`, потом `<S>` (или `File → Symlink`), добавить суффикс к имени, подтвердить.

## 19. Как узнать, сколько места занимает папка?
Выделить папку, нажать `<Ctrl+Space>` (или `Command → Show directory sizes`).

## 20. Как в Midnight Commander подключиться к FTP-серверу?
`Left → FTP link` (`<F9>`, `<L>`, `<P>`), ввести адрес вида `ftp://адрес_сервера`, нажать `OK`.

## 21. Как выполнить подключение к SFTP через SSH?
`Left → Shell link` (`<F9>`, `<L>`, `<S>`), ввести адрес сервера, нажать `OK`.

## 22. Как увидеть результат команды, введённой во встроенную командную строку?
Ввести команду в нижней строке, нажать `Enter`. Затем нажать `<Ctrl+O>`, чтобы временно свернуть Midnight Commander и увидеть вывод. Вернуться обратно — снова `<Ctrl+O>`.
